The Venusaur VMAX Starter Set (Japanese: スターターセットVMAX フシギバナ Starter Set VMAX: Fushigibana) and Blastoise VMAX Starter Set (Japanese: スターターセットVMAX カメックス Starter Set VMAX: Kamex) are a pair of Japanese-exclusive Standard Decks released for the Pokémon Trading Card Game on December 4, 2020.

Information

Each 60-card pre-constructed starter set premiers a Pokémon V and a Pokémon VMAX of the titular Pokémon and contains a fixed selection of cards. Each set comes packaged with a sheet containing 36 damage counters and 2 status markers, a player's guide, a coin, and a playmat featuring either Gigantamaxed Venusaur or Blastoise.

The VMAX Battle Triple Starter Set (Japanese: VMAX 対戦トリプルスターターセット) was also available from retailers on the same day. It includes a copy of both starter sets in addition to an updated version of the Charizard VMAX Starter Set, a sheet containing 36 damage counters and 2 status markers, a playmat featuring all three final evolutions of the Kanto first partner Pokémon, a deckbox, and four Holofoil S-P Promotional cards: Genesect, Heatran, Suicune, and Lugia.
